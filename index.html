What is this?

Airport Delays shows real-time airport delay data courtesy of a Firebase open data set that can be found at https://www.firebase.com/docs/open-data/. This data set comes directly from the FAA itself at http://services.faa.gov.

There's no need to refresh this page, it will automatically update if any data changes.

You can view the source code at https://github.com/speizerj/airportDelays.
